CMake Basics

To begin our journey, we are going to first make a CMakeLists.txt, and this will tell how to generate the Makefile or other files you would want generated.

There is a few things we need in order to create our project:

  • Setting the version of CMake: cmake_minimum_required(VERSION 3.10)
  • Naming your project and the version of it: project(<project-name> VERSION < version-number>)
  • Set the standard (you can choose to enforce it if you would like): set(CMAKE_CXX_STANDARD & lt;cpp-version>)
  • Set the source files: set(SRC_FILES ${PROJECT_SOURCE_DIR}/src/<file>.cpp)
  • Lastly, you will set the executable with all the source files and the name of the executable: add_executable(<name-of-exe>)

We will create our CMakeLists.txt and put the items listed above in it. Below is an example of mine. 


cmake_minimum_required(VERSION 3.10) 
set(PROJECT_NAME RayMaster)
# Project name and version.
project(${PROJECT_NAME} VERSION 0.1)

# Set the C++ Standard.
set(CMAKE_CXX_STANDARD 20)
set(CMAKE_CXX_STANDARD_REQUIRED true)

# Set default build to debug.
if (NOT CMAKE_BUILD_TYPE)
    set(CMAKE_BUILD_TYPE Debug)
endif()

# Add source files.
set(SRC_FILES ${PROJECT_SOURCE_DIR}/src/raymaster_driver.cpp)

# Add executable
add_executable(${PROJECT_NAME}  ${SRC_FILES})

Now in order to make this run we will need to create a directory, conventionally build, and run the following command cmake .. This will generate your makefile! Below is just an example of it being put in the terminal. 


mkdir build # create the build folder
cd build
cmake .. # from the build folder, run the cmake CLI command.

We've just generated our first Makefile!
